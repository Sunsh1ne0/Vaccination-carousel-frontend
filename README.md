## Setup steps

- Clone the repository
```
git clone https://github.com/Sunsh1ne0/Vaccination-carousel-frontend.git`
```
- Install the dependencies
```
npm install
```
- Build the project
```
npm run build
```

## Debugging
Run 
```
npm install
```
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
